**Overview:**  
Atlas is a **.NET 8 modular monolith backend** powering a mid-sized e-commerce platform.  
It integrates product catalogs, payments, and order management.  

**Key Features:**  
- CQRS with MediatR  
- Payment integration (Stripe, Razorpay)  
- GraphQL + REST API endpoints  
- Centralized logging with Serilog  
- Dockerized for easy deployment  

**Status:**  
🚧 In Development – Beta planned Q2 2025
