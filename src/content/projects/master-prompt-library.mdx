**Overview:**  
The definitive command center for professional AI interaction. This library eschews generic "copy-paste hacks" in favor of strict, constraint-based frameworks designed to extract maximum value from Large Language Models across multiple specialized disciplines.

**Domain Coverage:**  
- **Coding Engineering**: Specialized logic for API Design, System Architecture, DevOps, and Optimization.
- **Marketing & Branding**: Engineered direct response copy and data-driven strategic positioning frameworks.
- **Education & Academia**: लेसन planning, research synthesis, and complex topic deconstruction architectures.
- **Creative Arts**: High-fidelity prompts for scriptwriting, storytelling, and visual arts ideation.
- **Image Generation**: Precise directional frameworks for Midjourney, DALL-E, and Stable Diffusion.

**Architecture:**  
The repository utilizes a modular **Index/Example** structure to ensure clarity and repeatability.
- **`index.md`**: Houses the actual prompt logic, constraint rules, and variable definitions.
- **`eg.md`**: Provides high-quality, real-world examples of the prompt outputs for benchmarking.
