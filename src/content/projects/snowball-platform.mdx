**Overview:**  
**SnowBall** is a high-performance sports ecosystem designed to deliver real-time football data and deep community engagement. Built as a Turborepo monorepo, it integrates live match tracking, complex voting algorithms, and intelligent forecasting powered by a .NET backend and Microsoft Extensions AI.

**Key Features:**  
- **Real-time Match Updates**: Sub-second score tracking and live event synchronization.
- **AI Match Predictions**: Integrated AI models forecasting match outcomes based on historical performance and live data.
- **Community Engagement**: Weighted voting systems for match outcomes with real-time community sentiment visualization.
- **Admin Management Portal**: Comprehensive dashboard for match orchestration, event management, and platform-wide analytics.
- **Unified UI System**: Shared package architecture ensuring design consistency across the main portal and admin suite.

**Technical Stack:**  
- **Backend**: .NET Core with Microsoft Extensions AI integration.
- **Frontend**: Next.js 15, React 19, Tailwind CSS v4.
- **State Management**: TanStack React Query for optimized real-time data fetching.
- **Architecture**: Turborepo Monorepo for synchronized cross-application deployment.
