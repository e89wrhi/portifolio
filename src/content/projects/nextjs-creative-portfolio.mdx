**Overview:**  
Aurora is a **personal portfolio platform** designed for creatives and developers.  
It features a minimalist aesthetic, dark/light themes, and dynamic project pages powered by **MDX**.  

**Key Features:**  
- MDX-powered content management  
- Fully responsive with TailwindCSS  
- SEO-optimized with Open Graph metadata  
- Deployed on Vercel with global edge caching

✅ Live – [Visit Aurora Portfolio](https://aurora.dev)
