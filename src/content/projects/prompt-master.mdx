**Overview:**  
PromptMaster is a professional monorepo serving as a central hub for discoverable, reusable, and enterprise-grade AI templates. It streamlines the workflow for developers and creatives by providing a high-fidelity interface for prompt management and execution.

**Key Features:**  
- **Global Command Search**: Lightning-fast Cmd+K (or Ctrl+K) popup interface for instant template discovery across thousands of prompts.
- **Categorized Discovery**: Expertly organized across domains like Coding, Creative, Education, Marketing, and Productivity.
- **Advanced Filtering**: Nested architecture for granular topic drill-down and specific use-case targeting.
- **Interactive Template Editor**: Customize variables and prompt structures directly in the browser before deployment.
- **Standalone Prompt Pages**: Deep-linkable URLs with high-fidelity markdown rendering for every template.

**Technical Stack:**  
- **Framework**: Next.js 15 (App Router)
- **UI & Styling**: Tailwind CSS v4 & Lucide Icons
- **Management**: Turborepo for optimized monorepo performance
- **Language**: TypeScript
