**Overview:**  
Zenith is an **admin dashboard** template built with **Next.js 14**, designed for SaaS platforms.  
It features analytics, role-based access, and dark mode support.  

**Key Features:**  
- Interactive charts with Chart.js  
- Authentication & RBAC with NextAuth.js  
- Server Actions for CRUD APIs  
- Mobile-friendly layout
