**Overview:**  
Nova Shop Mobile is a **cross-platform shopping app** built with React Native Expo.  
It connects directly to the Horizon Shop backend to deliver a seamless customer experience.  

**Key Features:**  
- Browse catalog, add to cart, and checkout  
- Secure user authentication and profiles  
- Push notifications for promotions & order updates  
- Offline-first support with local caching  
- NativeWind-powered UI for a clean and modern look  

**Status:**  
🚀 Beta Testing – Available soon on TestFlight & Google Play Beta
